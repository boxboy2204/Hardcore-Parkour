# AGENTS.md

## Soul
This game is pure, intentional chaos: a clever, funny, high-energy parkour experience where failure is part of the joke and momentum is part of the fun. It is explicitly based on *The Office* universe and comedic style. Every agent run should preserve this spirit.

### Core vibe (always on)
- Prioritize fun, humor, and clever surprises over sterile realism.
- Keep the tone playful and witty, with quick comedic timing in flavor text, UI copy, and narrative beats.
- Treat mistakes, ragdolls, misses, and near-wins as opportunities for comedy, not punishment.
- Keep the experience joyful and approachable while still rewarding mastery.

### Creative direction
- Make ideas feel bright, vibrant, and expressive.
- Visual style should be colorful, energetic, and clearly inspired by *The Office*.
- Treat Dunder Mifflin (Scranton branch energy) as a core inspiration for world flavor, props, signage, naming, and event themes.
- Reference character archetypes and dynamics from Michael, Dwight, Jim, Pam, Stanley, Kevin, Angela, Creed, Kelly, Ryan, Oscar, Meredith, Toby, Darryl, and Andy.
- Favor punchy, readable writing with personality.
- Sprinkle small references, tidbits, and humorous one-liners throughout the experience to reward attentive players.

### Humor and writing rules
- Include short humorous quotes, quips, fake motivational lines, and cheeky meta-observations where appropriate.
- Regularly include recognizable *Office*-style references, including Dunder Mifflin nods, character catchphrase energy, and show-inspired quote moments.
- Use short quote snippets or paraphrased quote-style lines for inspiration in barks/UI text (for example: "Bears. Beets. Battlestar Galactica." or "That's what she said.").
- Use recurring comedic motifs and callbacks so jokes feel intentional, not random.
- Keep jokes light, inclusive, and game-appropriate.
- Balance comedy with clarity: players should always understand what to do next.

### Implementation intent
- When generating content, mechanics, UI text, dialogue, descriptions, or event copy, default to this Soul unless explicitly overridden by the user.
- If multiple valid options exist, choose the one that is more playful, clever, and memorable.
